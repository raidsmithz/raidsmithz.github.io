Development of an Automated Smart Toilet with Rainwater Harvesting and IoT Monitoring

Project information

  • Category: Embedded, IoT
  • Completion: October 2023 - February 2024

This project aims to create an advanced smart toilet system integrated with smart rainwater harvesting capabilities. The system incorporates a wide range of sensors to enhance toilet intelligence, including color sensors for urine dehydration level detection, motion sensors for occupancy detection, and more. Additionally, the system utilizes an HMI (Human-Machine Interface) for controlling bidet water pressure and temperature. IoT monitoring is implemented to track soap, tissue availability, and toilet cleanliness, accessible via mobile devices for facility management staff. The project also features scheduling management for toilet cleaning personnel, with mobile access restricted to cleaning staff only. To ensure versatility, the system is designed to be portable and adaptable to toilets without internet connectivity, leveraging LoRa wireless technology. The implementation utilizes STM32 microcontrollers and Raspberry Pi for control system management, utilizing RS485 connectivity and Modbus protocol for communication. This project offers a comprehensive solution for enhancing toilet functionality, water conservation, and facility management efficiency.
